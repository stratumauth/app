│ 🎓 关键技术点                                                                │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ 【架构模式】                                                                 │
│   • MVVM 模式 - 视图和逻辑分离                                               │
│   • 面板系统 - 模块化设计，易于扩展                                          │
│   • 依赖注入 - Autofac 容器管理                                              │
│                                                                              │
│ 【UI 框架】                                                                  │
│   • WPF (.NET 9.0) - 桌面应用框架                                            │
│   • Material Design 3 - 设计语言                                             │
│   • Frame 导航 - 面板切换                                                    │
│                                                                              │
│ 【关键组件】                                                                 │
│   • NavigationRail - 侧边栏导航（RadioButton + 自定义样式）                 │
│   • UserControl - 面板基类                                                   │
│   • ResourceDictionary - 颜色、样式、动画管理                               │
